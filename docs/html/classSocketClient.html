Detailed Description

Verzorgt de communicatie tussen de client en server via TCP-sockets.

Deze klasse stuurt en ontvangt gestructureerde berichten, waaronder sensorwaarden en besturingscommando's. Ondersteunt zowel echte als gesimuleerde data.

int SocketClient::receive_data ( )

Ontvangt en verwerkt inkomende berichten van de server.

Ondersteunt diverse datatypes zoals floats, bools, strings, enz. Bepaalt op basis van messageID welk subsysteem aangestuurd moet worden.

0 bij succes, 1 bij fout.

)

Stuurt een gegevensframe naar de server via de socket.

Het frame bestaat uit een message ID, metadata (vectorgrootte en datatype) en de daadwerkelijke data.

Parameters
messageIDUnieke identificatie voor het type bericht (bijv. 122 voor sensorwaarden).
vector_sizeAantal elementen in de vector (maximaal 15).
datatypeType van de data (0: uint32, 1: int32, 2: float, 3: bool, 4: string, 5: uint8).
dataPointer naar de ruwe data die verstuurd moet worden.
datasizeGrootte van de data in bytes (maximaal 1022).
Returns
0 bij succes, -1 als het verzenden mislukt.

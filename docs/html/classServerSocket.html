Verstuurt een dataframe naar een specifieke client.

Bouwt een bericht op volgens het gedefinieerde dataformaat en verstuurt dit via de juiste client socket.

Parameters
ip_addressHet IP-adres van de client.
messageIDEen identificatiecode voor het bericht (eerste byte).
vector_sizeAantal elementen in de datavector (maximaal 15).
datatypeHet datatype van de vector (0 = uint32_t, 1 = int32_t, 2 = float, 3 = bool, 4 = string).
dataPointer naar de databuffer.
datasizeGrootte van de databuffer (maximaal 1022 bytes).
Returns
0 bij succes, -1 bij fout (bv. IP niet gevonden of verzenden mislukt).

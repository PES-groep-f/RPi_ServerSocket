\doxysubsection{Detailed Description}
Verzorgt de communicatie tussen de client en server via TCP-\/sockets. 

Deze klasse stuurt en ontvangt gestructureerde berichten, waaronder sensorwaarden en besturingscommando\textquotesingle{}s. Ondersteunt zowel echte als gesimuleerde data.

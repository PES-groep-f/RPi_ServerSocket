\doxysubsection{Detailed Description}
Een eenvoudige LED-\/statusindicator met tekstlabel. 

Deze klasse toont een cirkelvormige LED die rood kleurt wanneer geactiveerd. Daarnaast wordt een tekstlabel weergegeven naast de LED. Klikken op de widget schakelt de LED uit indien deze aanstaat.
